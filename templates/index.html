Peer-to-Peer Plagiarism Detection for Google Classroom

Detect copying between students automatically. Compare assignments, analyze similarity using AI techniques, and help teachers identify plagiarism without manual checking.

Key Features

Google Classroom Integration

Secure login using Google OAuth and automatic access to courses, assignments, and student submissions.

Peer-to-Peer Plagiarism Detection

Detect similarity between students’ submissions using TF-IDF vectorization and cosine similarity.

PDF Text Extraction

Automatically extracts text from uploaded PDF assignments using Google Drive and PDF parsing.

Similarity Reports

Clear plagiarism reports showing student pairs and percentage similarity.

Student-wise Search

Filter submissions instantly using student IDs for faster inspection.

Teacher-Friendly Interface

Clean dashboard designed for teachers with no technical complexity.

About This Project

This project is designed to help teachers detect plagiarism between student submissions in Google Classroom. Unlike traditional plagiarism checkers that compare content against the internet, this system focuses on peer-to-peer similarity — the most common form of copying in classrooms.

Built using Flask, Google Classroom API, Google Drive API, PDF text extraction, and machine learning techniques, this tool automates a time-consuming task for educators.
